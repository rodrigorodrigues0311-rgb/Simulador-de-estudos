    ☁️ Passo 1 — Cloudflare Worker (proxy seguro)
    1
    Acesse workers.cloudflare.com → crie conta grátis
    2
    Clique em Create Worker → cole o conteúdo do arquivo worker.js
    3
    Settings → Variables → Add: Nome: ANTHROPIC_API_KEY · Valor: sua chave · marque Encrypt
    4
    Clique Save and Deploy — copie a URL (ex: simulado.user.workers.dev)
    🚀 Passo 2 — GitHub Pages
    1
    No index.html, linha com WORKER_URL, cole a URL do Worker
    2
    Suba os 4 arquivos: index.html · logo.png · manifest.json · sw.js no GitHub
    3
    Settings → Pages → Branch: main → Save
    4
    Link: seuusuario.github.io/simulado — compartilhe com todos! 🎉
